# Lucky Number 29

เว็บไซต์จองหมายเลขพร้อมระบบอัปโหลดสลิปและหน้า Admin เชื่อมกับ Supabase

## เริ่มใช้งาน

```bash
npm install
npm run dev
```

- หน้าลูกค้า: `/`
- หน้าผู้ดูแล: `/admin`

ก่อนเปิดใช้งานครั้งแรก ให้นำไฟล์ `supabase-setup.sql` ไปรันใน Supabase SQL Editor
